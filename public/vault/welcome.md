# Bem-vindo ao seu Obsidian Online! 🚀

Este é um visualizador de arquivos markdown e PDF simples e elegante.

## Funcionalidades
- [x] Visualização de arquivos **Markdown**
- [x] Visualização de arquivos **PDF**
- [x] Árvore de arquivos colapsável
- [x] Syntax highlighting para código
- [x] Tema escuro premiun

### Exemplo de código
```javascript
function hello() {
  console.log("Olá, Obsidian!");
}
```
